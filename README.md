# TP Gestión Colaborativa – Análisis de Ventas

## Integrante
- Giuliano De Vito

## Institución
UTN – Tecnicatura Universitaria en Programación (Modalidad a Distancia)

## Escenario
Escenario B – Análisis de ventas de una pequeña empresa

## Objetivo
Analizar un conjunto de datos de ventas para obtener indicadores básicos
y generar visualizaciones simples utilizando Python.

## Estructura del repositorio
- /datos: datasets utilizados en el análisis  
- /scripts: scripts de procesamiento y análisis  
- /resultados: gráficos y salidas generadas  

## Herramientas utilizadas

- Git y GitHub  
- Jira
